Jcpicker Program Kurulumu

Jcpicker programı bizim sayfalarımızda kullanacağımız; renkleri ekrandan yakalayıp, renk kodunu kopyaladıktan sonra kodda kullanılabilir hale getirmemizde önemli bir rol oynicak. Bir tek burada değil farklı çalışmalarınızda da kullanabilirsiniz. İndirdikten sonra karşınıza rar dosyası çıkıcaktır. Üstüne sağ tıklayıp, klasöre yada buraya ayıkla diyebilirsiniz.




Ben klasöre ayıklıyorum ve tıklıyorum.




Evet, program burda isterseniz masaüstüne alabilirsiniz. Tıkladığınızda karşınıza böyle bir ekran gelicek. Html yazan kısımda mauseumuzu ekranın herhangi bir köşesine getirip beğendiğimiz rengin üstünde Alt+x yaptığımızda bizim rengimizin farklı renk kodlarını ayarlicak ve değeri kopyalayıp kodumuzda kullanabileceğiz.
