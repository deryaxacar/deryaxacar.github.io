HTML Padding Nedir? Nasıl Kullanılır?

Merhaba arkadaşlar, dersimize hoşgeldiniz. Bu derste sizlere padding (içten boşluk) nedir? nasıl kullanılır, anlatacağım.


HTML Padding Nedir? Nasıl Kullanılır?

Padding, elementlerimizin uç noktaları arasındaki uzaklık olarak düşünülebilir, içten boşluk vermemizi sağlar. Örneklerle daha iyi anlicaksınız. Yeni bir sayfa oluşturuyorum, adına padding.html dedim. Bir textbox ekliyorum, < input type="text"> şeklinde. Çalıştırıp içine birşey yazıyorum. Padding özelliği yazdığımız yazı ile oluşturmuş olduğumuz textboxın çizgileri arasındaki boşluğu ayarlamamızı sağlicaktır.




Oluşturmuş olduğum textbox'a class="txtbox" şeklinde class veriyorum. Daha sonra yukarıdan style oluşturup, .txtbox{} diyorum. Özelliği vermek için {} arasına padding:10px; yazıyorum 10px çok fazla ama daha net görebilmeniz için yaptım. Eğer bütün kenarlardan değilde sadece soldan yapmak istiyorsanız padding-left vs kullanabilirsiniz. Ben tüm kenarlardan yapmak istediğim için sadece padding kullandım. Sağda görmüş olduğunuz gibi yazı ve textbox arasındaki boşluğu ayarladı.
