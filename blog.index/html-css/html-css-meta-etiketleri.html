HTML Meta Etiketleri Nelerdir?

Merhaba arkadaşlar, ilk derste sizlere HTML Meta etiketlerinden bahsedeceğim. Önceki yazıda indirdiğiniz Visual Studio Code'u açıyoruz oradan anlatacağım. Hazırsanız derse başlayalım.

HTML Meta Etiketleri Nelerdir?

Öncelikle Meta etiketleri nelerdir? bundan bahsedelim. Meta etiketleri, tarayıcıya sayfamız hakkında bilgi vermemizi sağlar, zaten diğer derslerde de göreceğiz. Şimdi projemize gelelim. Ders klosörümüzün üstüne sağ tıklayıp New File diyoruz ve yeni html sayfası açıyoruz. Açılan sayfanın uzantısının .html olmasına dikkat ediyoruz çünkü html sayfasından çalışacağız. Ben kendi sayfamın adına Layout diyeceğim. Peki neymiş bu Layout gelin beraber bakalım.




Html sayfamızı açtıktan sonra sayfa boş bir şekilde gelecektir. Her Html sayfasında aynı olan bir kısım vardır. İçinde sayfamızın bölümlerini, etiketlerini ve elementleri bulunduran bu kısıma Layout diyoruz ve ilk satıra ! yazıp onaylıyoruz böylece sayfamıza Layout eklemiş oluyoruz.




Eklenen kısım bu şekilde şimdi sizlere bu etiketlerin ne işe yaradıklarını anlatacağım.




İlk satır !DOCTYPE html yani bize sayfamızın dökümanı'nın ne olduğunu söylüyor. Bu bir html sayfası bize bunu belirtiyor.




2. Satır bize en (İngilizce), tr (Türkçe) gibi dilleri belirtiyor. Dili değiştirebilirsiniz de size kalmış hangi dilde çalışacağınız.




Şimdi gelelim sayfamızın asıl yerine. Html sayfaları 2 bölümden oluşur. Bu bölümler, head ve body kısımlarıdır. Head bizim bilgilerimizi tutarken body elementleri içinde bulundurur.




Öncelikle head'in içine bakalım. Charset sayfamızın Unicode'unu belirtir. peki nedir bu unicode?, unicode' un farklı sürümleri sayesinde İbranice, Arapca, Çince gibi kompleks anlaşılması zor olan dilleri kolayca dijital ortama aktarılmasını sağlar. Sadece diller değil, unicode sayesinde karmaşık karakterler ve semboller kolayca meydana getirilebilir. Kısacası anlaşılması zor olan dilleri dijital hale getirip bunu bilgisayarın anlayabileceği şekilde yazılmasını sağlar.





Şimdi birkaç meta etiketi de biz ekleyip devam edelim. Name kısmına yazdıklarımız bizim meta etiketimizin adı, content kısmına yazdıklarımız ise meta etiketimizin içeriği olacaktır. İlk meta etiketime description diyorum yani kelime anlamı olarak açıklama demektir. Content kısmının içine uzun uzun açıklama yapılırken alta eklediğim meta etiketi olan keywords (anahtar kelimeler yada anahtarlar demektir). Content'in içinde kısa ve öz kelimeler olmalıdır.





Son olarak bir meta etiketinden daha bahsedelim ve devam edelim. Outhor etiketi yani yazar etiketi, direkt content'in içine yazarın adını yazabilirsiniz. Gelelim title kısmına orası bizim sayfamızın başlığını belirtir. Yani tarayıcımızı çalıştırdığımızda sol üst kısımda bulunan sayfanın başlığıdır.
