CSS Nedir?

İngilizce kelime anlamı Cascading Style Sheets olarak kabül edilir, Basamaklanmış Stil Katmanları anlamına gelmektedir. CSS web sayfalarının görünüm ve görsellik açısından şekillendirmemizi sağlayan ve belli kuralları olan bir yazılım dilidir.


Bu dersimize kadar kullanmış olduğumuz CSS, İnline CSS olarak geçmekte ve çoğunlukla tercih edilmemektedir. Çünkü bir sayfada birden çok buton var diyelim hepsine teker teker o style ı kopyalamak çok uğraştırıcı ve bir o kadar da yorucu olurdu bu nedenle pek kullanılmamaktadır. Şimdi örneklerle anlatalım. Bir sayfa açalım, adına CssTypes.html diyorum ve bir buton oluşturuyorum. Buton oluşturmayı önceki dersimizde gösterdiğim şekilde yapıyorum bu sayfadaki birden fazla butona style vermek için ya ben gelip butonun içine verdiğim style'ı kopyalamam gerekir yada head'in içinde bu şekilde style etiketini kullanmam gerekir. Burdan tüm sayfanın style'ını ayarlayabiliriz.




Head'in içindeki style etiketi sadece o sayfada kullanılmaktadır. "Peki ya diğer sayfalarada uygulamak istersek?", bunun için dışarıdan yani tüm sayfaların dışından css vermemiz gerekir. Ders klasörümün üstüne gelip sağ tıklıyorum ve New Folder diyorum. Açtığım klasörün adına ben css dedim ve tekrardan açtığım klasörün üstüne sağ tıklıyorum, New File diyorum. Bu sefer açılan dosyamın adına ben style.css dedim dosya tipim .html değil .css olmak zorunda, hızlıca geçiyorum buraları ilerki derslerimizde görücez. Sayfamıza istediğimiz özellikleri verdik diyelim. "Diğer sayfadan nasıl çağırıcaz?" onu göstereceğim. Tabiki link ile çağıracağız. Link etiketimize href ekliyorum ve tırnakların içine / yazarak dosyamızın konumunu seçiyorum. Bunu kopyalayıp diğer sayfaların head kısmınada yapıştırabilirsiniz.
