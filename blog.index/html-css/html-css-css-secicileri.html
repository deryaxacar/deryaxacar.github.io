CSS Seçicileri (Selector) Nasıl Kullanılır?

Merhaba arkadaşlar, dersimize hoşgeldiniz. Bu derste sizlere CSS seçicilerinden bahsedeceğim, hazırsanız başlayalım.


CSS Seçicileri (Selector) Nasıl Kullanılır?

Selector, zaten kelime anlamı olarakta seçici demektir, sayfamızdaki bazı yada tüm elementleri seçerek özellikler verebilmekteyiz, bunlardan birkaçını kullanacağız. Öncelikle yeni bir sayfa oluşturup adına CssSelectors.html diyorum ve kod kısmına gelip layout ekliyorum. Daha sonra body'nin içine iki tane h1 ekleyip adımı yazıyorum.




Bu iki yazınında rengini aynı yapmak istiyorum. Yukarıdan style vermem gerek. Bütün h1'lerin rengini aynı yapmak için style'ın içine gelip h1 yazıp { } koyuyorum ve içine color: yazıp renk seçiyorum. Bu kod sayesinde bizim bütün h1lerimiz aynı renk olucaktır. Alta istersek bir milyon tane h1 yazalım, yukarıdan h1'lere style verdiğimiz için hepsinin rengi aynı olucaktır.




Şimdi bunlara ayrı ayrı renk verelim. Bunun için id ve class kavramlarını kullanıcağız. Birine class diğerine id veriyorum. Yukarıdaki h1'in classı hred, alttaki h1'in idsi hdr1 olsun. Class'ı kullanmak için style kısmına geliyorum ve . koyup h1'in classına verdiğim adı yazdıktan sonra { } koyup içine renk veriyorum. İkinci h1'e renk vermek için alta inip # koyuyorum ve id yazıyorum. yani class için nokta id için # kullanıyoruz.




Şimdi style ı dışardan verelim. sol taraftaki dosyalar kısmına geliyorum ve önceden açmış olduğum css dosyamın içine tıklayıp, style.css sayfasını açıyorum ve önceki sayfadaki style kodunu kopyalayıp bu sayfaya yapıştırıyorum.




Daha sonra CssSelectors.html sayfasına dönüp link ile diğer sayfayı çağırıyorum. < link rel="stylesheet" href="css/style.css" > şeklinde. Çalıştırdığınızda yine aynı sonucu vericektir.




Son olarak *'ı anlatayim. Style kısmına * yazıp { } açtığımızda içine verdiğimiz özellikler tüm sayfayı kapsayacaktır. Yani sayfadaki sadece bütün h1'ler değil body kısmını ve diğerlerinide kapsayacaktır. Ona göre kullanırsınız.
