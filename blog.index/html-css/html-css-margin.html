HTML Margin Nedir? Nasıl Kullanılır?

Merhaba arkadaşlar, dersimize hoşgeldiniz. Bu derste sizlere margin (dıştan boşluk) nedir? nasıl kullanılır, anlatacağım.


HTML Margin Nedir? Nasıl Kullanılır?

Margin, elementimizin dışarısından sağa,sola,yukarı yada aşşağıya verilebilen boşluklardır. Yeni bir sayfa oluşturuyorum, adına margin.html dedim. Örnek olarak < input type="button"> şeklinde iki buton ekliyorum. Margin sayesinde bu iki butonu birbirinden uzaklaştırmak için boşluklar vericez. Önce böyle çalıştırıp görelim.




Farkettiğiniz üzere butonlarımız sayfanın tam üstüne, tam sağına yada tam olarak birbirlerine yapışık değiller. Bunun nedeni sayfamız oluşturulurken kendiliğinden biraz margin ve biraz padding veriyor. Bunu sıfırlamak için style kısmına gelip *{} koyarak tüm elementleri seçip içinde padding ve margin özelliklerini bu şekilde sıfırlayabilirsiniz.




Butonlarımıza dönücek olursak, 2. butona class="btn" şeklinde class verip style kısmına geliyorum. Daha sonra .btn{} şeklinde oluşturmuş olduğum class'a sağ tarafa gitmesi için soldan boşluk vermem gerektiğinden margin-left: 200px; yazarak soldan margin'ini 200px olarak ayarlamasını istiyorum. Görmüş olduğunuz gibi çalıştırdığımda butonlar arası boşluk bu şekilde olucaktır.
