ASP.NET Nedir? Nasıl Kullanılır?

Merhaba arkadaşlar. Bu yazıda sizlere ASP.NET nedir ve nasıl kullanılır bunlardan bahsedeceğim.

ASP.NET Nedir?

"ASP.NET nedir?" sorusunun cevabı, Microsoft tarafından geliştirilmiş olan ve aktif sunucu sayfaları olarak da bilinen Asp.Net, bünyesinde birtakım kodları, nesneleri de barındırmaktadır. Bu kodlar ve nesneler sayesinde istenilen komutlar kullanılarak oluşturulmaktadır. Web uygulama, web sitesi, framework geliştirme çatısıdır. Normal ASP'nin üzerine bazı geliştirmeler yapılarak oluşturulan Asp.net, Web sayfaları, Web uygulamaları, XML gibi Web servisleri hazırlanmasında önemli bir rol oynar. ASP.NET bir programlama dili değildir ve birçok amaç için kullanılabilir.



Nasıl Kullanılır?

"ASP.NET nasıl kullanılır?" sorusuna gelirsek, bir web uygulaması hazırlayabilmek için .Net dillerinden olan C# ve bunun gibi dilerden biri kullanılabilir. Asp.net ile web uygulaması, web servisleri geliştirmek için Visual Studio kullanılır. bunun yanı sıra web sitelerini geliştirmekte kullanılabilir ilerki derslerde nasıl kullanılabileceği hakkında sizlerle çalışmalar paylaşacağım ve nasıl kullanıldığını sizlerle öğretmiş olacağım.dersler kısmından ilk derse ulaşabilirsiniz.

ASP.NET İle Neler Yapılabilinir?

aklınızdan "ASP.NET ile neler yapılabilir?" diye geçiriyorsanız size anlatayım, hazır web şablonlarını kişilleştirip kendi veritabanlarını oluşturarak erişim sağlanabilir genellikle E- ticaret yada bazı firmalar tarafından sıkça kullanılmaktadır bu işte kendinizi yeterince geliştirdiğiniz taktirde işi ticarete döküp para kazanmaya başlayabilirsiniz günümüzde bir çok firma yada bu işlerle ilgilenen kişiler bu tür yollarla para kazanmakta.
